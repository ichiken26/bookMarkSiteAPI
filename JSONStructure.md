私は現在、ブックマークサイトをJSONで一元管理しているブックマークサイトを作っています。  
現在、JSONを格納しているフォルダに各項目ごとのURLをまとめたサイトがあります。  

```json
[
    {
        "name": "urlName",
        "url": "https://example.com"
    },
]
```

各JSONの構造はこのようになっています。  
さて、こちらを静的サイトではなくAPI化したいというのが現在の目的です。  
CRUDの仕様表とAPI設計書を`./APIspec.md`に出力してください。  

Cloudflare WorkersにこのAPIはデプロイし、DBとしてはD1を作成する予定なのでそれを前提にお願いします。
また、CMS上で並び替えも可能にするので、並び替えもお願いします。

- カテゴリ一覧
- カテゴリごとのID・名前・並び順
- 各カテゴリIDに紐づいたブックマークID
- ブックマークIDに紐づいたname・url・並び順

これらが必要なGETだと感じます。
